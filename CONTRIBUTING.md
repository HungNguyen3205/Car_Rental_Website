# Hướng dẫn đóng góp (Contributing Guide)

Chào mừng bạn đã đến với dự án **Midnight Luxury**! Chúng tôi rất trân trọng mọi sự đóng góp từ cộng đồng để dự án ngày càng hoàn thiện hơn.

## 🚀 Cách thức đóng góp

1.  **Fork** dự án này về tài khoản GitHub của bạn.
2.  **Clone** dự án đã fork về máy cá nhân:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Car_Rental_Website.git
    ```
3.  Tạo một **Branch** mới cho tính năng hoặc bản sửa lỗi của bạn:
    ```bash
    git checkout -b feature/AmazingFeature
    ```
4.  **Commit** các thay đổi của bạn:
    ```bash
    git commit -m 'Add some AmazingFeature'
    ```
5.  **Push** lên branch của bạn:
    ```bash
    git push origin feature/AmazingFeature
    ```
6.  Mở một **Pull Request** trên GitHub.

## 📝 Quy chuẩn Code

-   Sử dụng Tiếng Việt hoặc Tiếng Anh rõ ràng trong comment và commit message.
-   Tuân thủ cấu trúc thư mục hiện có (BE, FE, AI_Assistant).
-   Đảm bảo code đã được test kỹ trước khi tạo PR.
